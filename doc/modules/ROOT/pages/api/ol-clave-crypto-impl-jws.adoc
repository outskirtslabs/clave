= ol.clave.crypto.impl.jws

[#jws-alg]
== jws-alg





Return the JWS `alg` header value for a key.

| Key Type | Result    |
|----------|-----------|
| P-256    | "ES256" |
| P-384    | "ES384" |
| Ed25519  | "EdDSA" |
| RSA      | "RS256" |

[.api-source]
link:https://github.com/outskirtslabs/clave/blob/main/src/ol/clave/crypto/impl/jws.clj#L123-L132[source,window=_blank]

'''

[#sign]
== sign





Sign data bytes, return signature bytes in JWS format.

For ECDSA, returns R||S concatenated (not DER).
For EdDSA and RSA, returns raw signature bytes.

[.api-source]
link:https://github.com/outskirtslabs/clave/blob/main/src/ol/clave/crypto/impl/jws.clj#L148-L153[source,window=_blank]
